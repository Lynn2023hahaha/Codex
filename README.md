# Special for Anna

A bilingual measurement-units learning tool for children.

## Features

- English and Chinese language switch
- Length, weight, capacity, time, temperature, and area lessons
- Interactive unit converter
- Quick practice questions with scoring
- Static frontend only: no backend required

## Deploy

This project can be deployed as a static site on Cloudflare Pages, Netlify, Vercel, or GitHub Pages.

For Cloudflare Pages:

1. Push this project to GitHub.
2. Create a new Cloudflare Pages project.
3. Connect the GitHub repository.
4. Use no build command.
5. Use the project root as the output directory.
